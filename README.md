# Website Profile - Mochammad Naufal Bagaskara

Website profile profesional untuk Programmer, Konsultan IT, dan Desain Grafis dengan tema merah maroon.

## Instalasi

```bash
npm install
```

## Menjalankan Server

```bash
npm start
```

Website akan berjalan di http://localhost:3000

## Struktur Project

```
├── server.js                 # Server Node.js dengan Express
├── package.json             # Konfigurasi project
├── public/
│   ├── index.html          # HTML utama
│   ├── css/
│   │   ├── main.css        # Import semua CSS
│   │   ├── variables.css   # Color variables
│   │   ├── base.css        # Base styles
│   │   ├── cursor.css      # Custom cursor
│   │   ├── particles.css   # Particles background
│   │   ├── header.css      # Header & navigation
│   │   ├── hero.css        # Hero section
│   │   ├── about.css       # About section
│   │   ├── services.css    # Services section
│   │   ├── contact.css     # Contact section
│   │   ├── footer.css      # Footer
│   │   ├── animations.css  # Animations
│   │   └── responsive.css  # Responsive design
│   └── js/
│       ├── cursor.js       # Custom cursor logic
│       ├── particles.js    # Particles animation
│       ├── scroll.js       # Smooth scroll & navbar
│       ├── animations.js   # Scroll animations & counter
│       ├── typing.js       # Typing effect
│       └── cards.js        # Card hover effects
```

## Fitur

- Responsive design
- Custom cursor dengan glow effect
- Particles background animation
- Smooth scroll navigation
- Typing animation
- Counter animation
- Card hover effects dengan glow
- Gradient orbs floating animation
- Glitch text effect
- Tema merah maroon yang elegan

## Customisasi

### Mengubah Warna
Edit file `public/css/variables.css` untuk mengubah color scheme

### Mengubah Konten
Edit file `public/index.html` untuk mengubah teks dan konten

### Mengubah Animasi
Edit file di folder `public/js/` untuk mengubah behavior animasi
